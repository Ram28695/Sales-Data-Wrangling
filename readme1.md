
## Overview
This project demonstrates how to analyze and visualize sales data using **Excel PivotTables and PivotCharts**.  
The goal is to build a professional dashboard that highlights key business insights such as revenue trends, top customers, category contributions, and regional performance.


## Project Files
- `Sales_Dataset.csv` → Raw dataset  
- `Sales_Dataset_Cleaned.csv` → Cleaned dataset after wrangling  
- `Data_Dictionary.xlsx` → Documentation of fields and transformations  
- `prep1.py` → Python script used for initial data wrangling  
- `Sales_Dashboard_Final.xlsx` → Completed Excel dashboard with PivotTables and charts  
- `README.md` → Project documentation (this file)

---

## Steps Performed
1. **Data Wrangling**
   - Cleaned raw dataset (handled missing values, standardized formats).
   - Documented changes in the Data Dictionary.

2. **PivotTable Creation**
   - Monthly Revenue Trend (Line Chart).
   - Top 5 Customers by Revenue (Bar Chart).
   - Revenue Share by Category (Pie Chart).
   - Average Order Value by State (Bar Chart).

3. **Dashboard Assembly**
   - Combined all charts into a single Dashboard sheet.
   - Added KPI cards:
     - Total Revenue  
     - Total Profit  
     - Average Order Value  
     - Total Orders  

4. **Final Formatting**
   - Applied consistent colors, borders, and layout.
   - Added a professional header: *Sales Performance Dashboard*.



## Key Insights
- Revenue shows a clear monthly trend with seasonal peaks.  
- Top 5 customers contribute a significant share of total sales.  
- Electronics category dominates revenue share.  
- Certain states show higher average order values, indicating premium markets.



##  How to Use
1. Open `Sales_Dashboard_Final.xlsx` in Excel.  
2. Explore the Dashboard sheet for charts and KPIs.  
3. Use PivotTables to drill down into specific categories or customers.
